# Meeting Welcome

A standalone onboarding page for a Webfuse-style meeting space. Guides a first-time visitor through a 5-step tour with hand-drawn arrows pointing at different parts of the UI.

- **Live:** https://aogz.github.io/meeting-welcome/
- Single `index.html`, no build step.
- FontAwesome + Manrope loaded from CDNs.

## Tour steps

1. Participants (top-left)
2. Tabs — `+` for a new tab, click a tab to edit its URL
3. Video & Chat (top-right, offset)
4. Settings & Exit (top-right corner)
5. Apps, extensions & space settings, with example use cases
